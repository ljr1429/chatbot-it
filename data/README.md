# 📁 data 폴더

이 폴더에 회사 PDF 문서를 추가하세요.

## 사용 방법

1. 회사 문서(PDF)를 이 폴더에 복사
2. `npm run index-pdfs` 실행
3. Supabase에 벡터 데이터 자동 저장

## 권장 문서 종류

- 서비스 소개서
- 제품 카탈로그
- 사용 가이드
- FAQ 문서
- 가격표
- 정책/약관

## 주의사항

- PDF 파일만 지원됩니다
- 한글 파일명 사용 가능
- `.gitignore`에 의해 PDF 파일은 Git에 포함되지 않습니다
- 실제 배포 시 PDF 파일을 직접 추가해주세요
